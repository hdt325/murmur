Murmur runs locally on your computer. Start the server to connect:

1
Install prerequisites npm i -g @anthropic-ai/claude-code uv tool install voicemode
2
Clone and install git clone https://github.com/hdt325/murmur.git cd murmur && npm install
3
Start the server npx tsx server.ts
4
Start voice services (in separate terminals) voicemode service whisper start voicemode service kokoro start
